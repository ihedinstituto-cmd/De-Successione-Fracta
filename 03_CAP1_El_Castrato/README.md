# Capítulo I — El Castrato que Sabía Demasiado

**Atto Melani (1626–1714)**

Castrato, compositor y agente de inteligencia de Luis XIV. Red de espionaje musical entre Roma, Florencia, París y Madrid. Conexión con el Archivio di Stato di Firenze (Mediceo del Principato, filze 5421–5425) y el Archivio di Stato di Pistoia.
